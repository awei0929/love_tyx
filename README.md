# Arco Design Pro

## 快速开始

```js
// 初始化项目
npm install

// 开发模式
npm run dev

// 发布测试环境
npm run build:test

// 发布灰度环境
npm run build:gray

// 发布正式环境
npm run build:prod
```

## setting.json

```typescript
{
  "prefixCls":string, //arco design 组件 css 类名前缀
  "themeColor": string,//主题色
  "navbarHeight":number, //顶部栏高度
  "menuWidth": number //侧边栏宽度
}
```
